Here are some of the projects we've been working on at Sequoia High School.

Sound Board

We created an app that had a collection of pictures. When the pictures are tapped, a sound is played. If the picture is tapped again, the sound pause. This was our introduction to utilizing media in App Inventor and using true/false logic.

Paint Pot

We create an app that is very similar to Snap Chat. The students created three buttons that each correspond to a different color. When the color is tapped, the user can begin drawing across the screen with the color that was selected.

Image Sprites

Image sprites builds on top of Paint Pot and allows the user to add stamps to be place over the existing canvas.
